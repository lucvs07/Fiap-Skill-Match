# Fiap Skill Match

## 📌 Descrição do Problema

O processo de formação de grupos para Iniciação Científica (IC) pode ser desafiador, tanto para alunos quanto para mentores, devido à dificuldade de encontrar oportunidades alinhadas aos interesses, habilidades e disponibilidade de cada participante.

## 🚀 Solução Proposta

O Fiap Skill Match é um sistema que conecta alunos e mentores, facilitando a publicação, busca e participação em projetos de IC. O app permite cadastrar alunos, mentores e projetos, além de gerenciar interesses, vagas e contatos de forma centralizada e transparente.

## 🛠️ Tecnologias Utilizadas

- Python 3
- Persistência de dados em arquivos JSON
- Execução via terminal/console

## ⚙️ Como Executar

1. Certifique-se de ter o Python 3 instalado.
2. Clone o repositório ou baixe os arquivos do projeto.
3. No terminal, navegue até a pasta do projeto.
4. Execute o comando:
   ```bash
   python main.py
   ```

## 📂 Estrutura do Projeto

- `main.py` — Código principal do sistema
- `alunos.json` — Dados persistentes dos alunos
- `mentores.json` — Dados persistentes dos mentores
- `projetos.json` — Dados persistentes dos projetos

## 🧩 Funcionalidades Implementadas

- Cadastro de alunos e mentores
- Publicação de projetos de IC
- Mural de projetos com filtros e busca
- Diretório de mentores disponíveis
- Manifestação de interesse em projetos
- Visualização de interessados
- Alteração de status dos projetos
- Persistência local dos dados
